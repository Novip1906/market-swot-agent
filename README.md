# Market Agent (Рыночный Агент)

**Market Agent** — это Telegram-бот на языке Go, предназначенный для проведения рыночного анализа компаний. Бот объединяет возможности поиска в реальном времени через **Tavily Search API** и аналитические способности современных языковых моделей (LLM, например GPT-4o) для автоматической генерации SWOT-анализа.

## 🚀 Основные возможности

- 🔍 **Поиск информации в реальном времени**: Использование Tavily для получения самой актуальной информации о компаниях и рынках.
- 📊 **Автоматический SWOT-анализ**: Генерация отчетов о сильных и слабых сторонах, возможностях и угрозах на основе найденных данных.
- ⚙️ **Гибкая настройка**: Поддержка различных LLM-провайдеров и легкая конфигурация через переменные окружения.

## 🛠 Стек технологий

- **Язык**: Go (Golang)
- **Библиотеки**: 
  - `telebot` для работы с Telegram Bot API.
- **Интеграции**:
  - **Tavily API** для умного веб-поиска.
  - **OpenAI API** для анализа текста.

## 📦 Установка и запуск

### Предварительные требования

- Установленный [Go](https://go.dev/dl/) (версии 1.21 или выше).
- Telegram Bot Token (получить у [@BotFather](https://t.me/botfather)).
- API ключ Tavily (получить на [tavily.com](https://tavily.com/)).
- API ключ OpenAI (или другого LLM провайдера).

### Шаги по установке

1. Клонируйте репозиторий:
   ```bash
   git clone https://github.com/philippschepnov/market-agent.git
   cd market-agent
   ```

2. Создайте файл `.env` в корневой директории и заполните его

3. Запустите бота:
   ```bash
   go run main.go
   ```

## ⚙️ Переменные окружения

Настройте следующие переменные в вашем `.env` файле:

| Переменная | Описание |
|------------|----------|
| `TELEGRAM_APITOKEN` | Токен вашего Telegram-бота |
| `OPENAI_API_KEY` | API ключ для доступа к LLM (OpenAI или аналог) |
| `TAVILY_API_KEY` | API ключ для Tavily Search API |
| `OPENAI_API_BASE` | (Опционально) Базовый URL для API, если используете прокси или другой провайдер |
| `MODEL_NAME` | Название используемой модели |

## 💡 Использование

1. Запустите бота и отправьте команду `/start`.
2. Введите название компании (или домен), которую вы хотите проанализировать.
3. Дождитесь, пока агент соберет данные и сформирует SWOT-отчет.
